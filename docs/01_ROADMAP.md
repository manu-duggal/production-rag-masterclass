# Production RAG Masterclass Roadmap

> This document tracks the complete roadmap for the Production RAG Masterclass.
>
> The roadmap may evolve as new concepts are introduced, but every change should improve the student's understanding rather than simply adding more topics.

---

# Overall Goal

Learn Retrieval-Augmented Generation (RAG) like a Production AI Engineer.

By the end of this roadmap, the student should be able to:

- Design production RAG systems.
- Build production-ready RAG applications.
- Debug retrieval issues.
- Evaluate retrieval quality.
- Explain engineering trade-offs.
- Answer interview and system design questions confidently.
